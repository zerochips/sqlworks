SQL 언어
- DDL : Data Definition Language(데이터 정의어)
	(Create, Alter, Drop)
- DML : Data Mainpulation Language(데이터 조작)
	(Insert, Select, Deleta -> Commit, Rollback)
- DCL : Data Control Language(데이터 제어어)
	(Grant, Revoke)
